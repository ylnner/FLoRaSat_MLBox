Satellite Mobility

Overview

The Mobility folder provides orbital mechanics integration using NORAD TLE (Two-Line Element) data to simulate realistic satellite trajectories.

Architecture

Mobility/
└── INorad: Interface for TLE-based orbital propagation

Key Components:

  • INorad: Base module for NORAD SGP4/SDP4 orbit propagation
    • Loads TLE files for satellite ephemeris
    • Computes position (lat/lon/alt) at simulation time
    • Calculates elevation and azimuth to reference points
    • Uses libnorad for SGP4 algorithms

Usage

Satellites and stations reference INorad implementations for position updates:

  • Sat_Mob_NoradA: Satellite mobility using TLE data
  • Sat_Mob_SatelliteMobility: Generic LEO mobility
  • Sta_Mob_StationMobility: Ground station fixed positions

Implementation Details

  • Translates simulation time to Julian dates
  • Downloads/caches TLE files from online sources
  • Updates positions at configurable intervals (e.g., updateInterval = 1s)
  • Supports coordinate conversions (ECI ↔ Geodetic)
