Detailed Description

Base class for MAC protocol implementations without INET MacProtocolBase dependency.

This class provides essential MAC layer functionality similar to INET's MacProtocolBase but is independent of INET's lifecycle and operational framework, making it suitable for custom implementations in the FLoRaSat simulator.
