createTransmission method which is called when a transmission is made between a source and destination.

The transmitter must create a transmission model object which encapsulates all of the transmission details. The coordinates are encapsulated to the transmission model so that the required INET models can use the real coordinates to calculate distance over the 2D OMNeT++ positions.
