Medium Models

Overview

The Medium folder provides specialized radio medium implementations for different link types in the satellite network.

Architecture

Medium/
└── DSL/: Device-to-Satellite Link medium configuration

Key Components:

  • DSL/: Radio medium parameters for terminal-to-satellite communications
    • Path loss models
    • Propagation delay
    • Obstacle modeling
    • Background noise

Usage

Medium types are configured per radio interface:

*.radioMedium.typename = "LoRaRadioMedium"
*.radioMedium.pathLossType = "LoRaLogNormalShadowing"

Implementation Details

  • Each medium type defines propagation characteristics
  • Integrates with INET's IRadioMedium interface
  • Coordinates with Channel and Transceiver modules
  • Supports heterogeneous link types (DSL, GSL, ISL)
