Terminal Physical Layer

Overview

The terminal physical layer implements LoRa radio interfaces for device-to-satellite communications, managing transmission parameters and radio state.

Architecture

_10_Physical/
└── Ter_Phy_LoRaRadio: LoRa radio module with configurable SF/BW/CR

Key Components:

  • Ter_Phy_LoRaRadio: Extends INET radio base with LoRa-specific parameters
    • Spreading Factor (SF7-SF12)
    • Bandwidth (125/250/500 kHz)
    • Coding Rate (4/5, 4/6, 4/7, 4/8)
    • Transmit Power (configurable)

Radio States

  • IDLE: Listening for preambles
  • TRANSMITTING: Sending uplink data
  • RECEIVING: Processing downlink from satellite

Configuration

*.terminal[*].radio.typename = "Ter_Phy_LoRaRadio"
*.terminal[*].radio.transmitter.power = 14dBm
*.terminal[*].radio.receiver.sensitivity = -130dBm

Implementation Details

  • Interfaces with LoRaTransmitter and LoRaReceiver
  • Supports dynamic SF adaptation (future work)
  • Coordinates with MAC for channel access
  • Reports RSSI and SNR to upper layers
