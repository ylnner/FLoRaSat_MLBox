Ground Station Node

Overview

The Ground Station represents fixed infrastructure providing satellite-to-terrestrial network connectivity. It acts as gateway between the satellite constellation and the Internet/backend servers.

Architecture

Sta (Compound Module)
├── _10_Physical/: High-gain radio for GSL
├── _20_Data_Link/: Ground station MAC protocol
├── _30_Network/: Gateway routing and tunneling
├── _40_Application/: Data aggregation and forwarding
├── _50_Energy/: Powered infrastructure (unlimited)
└── _60_Mobility/: Fixed geographic position

Key Components:

  • Sta.ned: Compound module with GSL radio interface
  • Sta.h/cc: Container module for ground station node
  • Gateway Functions: Aggregates satellite traffic, forwards to servers

Packet Flow

  1. Downlink: Satellite (GSL) → Ground Station → Internet/Server
  2. Uplink: Server/Internet → Ground Station → Satellite (GSL)

Implementation Details

  • Fixed geographic locations (high-latitude, equatorial, etc.)
  • High TX power and low noise figures
  • Tracks satellite passes and schedules communication windows
  • Aggregates traffic from multiple satellites
