Constellation Creator

Overview

The Constellation Creator dynamically generates satellite constellations with Walker Delta patterns, configuring orbital parameters and inter-plane spacing.

Architecture

ConstellationCreator
├── Creates satellite nodes at initialization
├── Distributes satellites across orbital planes
└── Sets RAAN and mean anomaly for each satellite

Key Components:

  • ConstellationCreator: Module that spawns satellite nodes based on configuration parameters
    • satCount: Total number of satellites
    • planeCount: Number of orbital planes
    • satsPerPlane: Satellites per plane
    • inclination: Orbital inclination (degrees)
    • altitude: Orbit height (km)
    • interPlaneSpacing: Relative phasing between planes

Usage

Configure constellation in .ini:

*.constellationCreator.satCount = 66
*.constellationCreator.planeCount = 6
*.constellationCreator.satsPerPlane = 11
*.constellationCreator.inclination = 86.4
*.constellationCreator.altitude = 780

Implementation Details

  • Calculates orbital elements (RAAN, mean anomaly) for uniform distribution
  • Interfaces with Sat_Mob_NoradA to set initial positions
  • Supports Walker Delta and Flower constellation patterns
  • Creates satellites during initialize() stage
