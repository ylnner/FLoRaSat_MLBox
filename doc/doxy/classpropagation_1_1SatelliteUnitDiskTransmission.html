The SatelliteUnitDiskTransmission class is a model representing the Unit Disk Transmission of a satellite.

Unlike the INET UnitDiskTransmission, this model encapsulates start and end position coordinates so that the propagation model can use the actual long/lat/altitude positions of a ground-station/satellite over the 2D OMNeT++ canvas positions.
